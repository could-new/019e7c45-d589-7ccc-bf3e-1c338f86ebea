# تطبيق تجريبي

هذا التطبيق هو تطبيق تجريبي مبني باستخدام Flutter. تم تحديث الواجهة لتدعم اللغة العربية بالكامل، مع تعديل اتجاه العرض ليكون من اليمين إلى اليسار (RTL).

## المميزات
- واجهة مستخدم باللغة العربية.
- دعم العرض من اليمين إلى اليسار (RTL).
- عداد بسيط يوضح عدد مرات الضغط على الزر.

## كيفية التشغيل
تأكد من تثبيت بيئة عمل Flutter، ثم قم بتشغيل الأمر التالي:
```bash
flutter run
```
